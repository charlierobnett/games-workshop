# Games Workshop

This is where our games live. Each folder under `games/` is a game we built together.

## Layout

```
games-workshop/
├── shared/
│   └── assets/        ← pixel packs, sprites, audio we can reuse across games
└── games/
    └── <game-name>/   ← one folder per game
```

## How we build

Every game follows the same flow:

1. **Talk about it** — what do you want to make?
2. **Write a spec** — Claude asks questions; you answer
3. **Build milestone 1** — play it
4. **Build milestone 2** — play it
5. **Build milestone 3** — play it
6. **Ship it** — push to GitHub, get a live link

Each milestone is playable on its own. We don't move on until the current one feels good.

The full step-by-step lives in `c:/AI-OS/02_DOMAIN_OS/HOME_OS/playbooks/game-build-session.md`.
