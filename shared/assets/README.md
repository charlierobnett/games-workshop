# Shared Assets

Pixel packs and sprite sheets shared across games in this repo.

## How to add a pack

1. Download from a license-clean source (see below)
2. Unzip into this folder under a clearly-named subdirectory: `<source>-<theme>/`
   - Example: `kenney-pixel-platformer/`, `kenney-tiny-dungeon/`
3. Keep the original license file alongside the assets

## Recommended sources

| Source | License | Notes |
|--------|---------|-------|
| [Kenney](https://kenney.nl/assets) | CC0 (public domain) | Default choice. No attribution needed. |
| [OpenGameArt](https://opengameart.org) | Mixed — check each pack | Big variety; read the license |
| [itch.io free assets](https://itch.io/game-assets/free) | Mixed | Lots of indie packs |

## First pack to download (manual step)

Start with **Kenney's Pixel Platformer** or **Tiny Dungeon** pack — they have characters, environments, and UI elements that cover most beginner game types.

Download URL: https://kenney.nl/assets/pixel-platformer

Unzip into `kenney-pixel-platformer/` and we're ready to build.
